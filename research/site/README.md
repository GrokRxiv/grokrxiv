# GrokRxiv research site (local-only)

This is a standalone Next.js viewer for `research/*.html` artifacts. It is NOT deployed and has no coupling to `apps/web/`.

## Run

```sh
cd research/site
pnpm install
pnpm dev
# open http://localhost:3100
```

The HTML artifacts are generated by `research/_template/build.mjs` (Track B of FP6). If you don't see new content, re-run the build script or wait for the Stop-hook to trigger.
