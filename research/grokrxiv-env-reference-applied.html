#grokrxiv env-var reference — applied

Env vars consumed by the orchestrator binary (grokrxiv / grokrxiv-orchestrator) and the Next.js web tier. Layered config order is: CLI flags > ENV > TOML profile > built-in defaults. The CLI's --profile <name> and --config <path> flags pick the TOML file/profile that ENV then overrides.
